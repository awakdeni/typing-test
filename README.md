# Typing Test

A simple and fast typing speed test application built with Next.js 16.  
This project is designed to simulate a real typing experience similar to popular platforms, with a focus on speed, accuracy, and smooth user interaction.

---

## Features

- Real-time typing speed (WPM) calculation  
- Accuracy tracking (correct and incorrect characters)  
- Per-word typing system with active word highlighting  
- Visual feedback for correct and incorrect input  
- Auto scroll to the current active word  
- Responsive design for desktop and mobile  
- Lightweight and fast performance  

---

## Project Purpose

This project was built to demonstrate skills in:

- Handling keyboard input efficiently  
- Building real-time systems without lag  
- Creating interactive and responsive user interfaces  
- Writing clean and maintainable code  

---

## Tech Stack

- Next.js 16  
- React  
- CSS / Tailwind (optional)  

---

## How It Works

- User input is captured using a hidden input field  
- Text is rendered word by word  
- Each character is validated in real-time  
- WPM is calculated based on typed characters
